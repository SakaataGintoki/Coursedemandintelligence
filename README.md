# Course Demand Intelligence

## Data-Driven Course Demand Prediction and Resource Planning System

**University:** RJ College of Arts Science & Commerce  
**Student:** Rishab Singh (10441)

---

## 🚀 Overview
A machine learning system that predicts student enrollment for 20 university courses across 4 departments.

This system helps academic administrators:
- Plan course sections
- Allocate faculty efficiently
- Optimize classroom usage
- Forecast budgets before semester start

---

## 📂 Project Structure

| File | Description |
|------|------------|
| `course_demand_prediction.py` | ML pipeline: data generation, training, evaluation |
| `dashboard.py` | Streamlit dashboard with interactive analytics |
| `requirements.txt` | Required dependencies |
